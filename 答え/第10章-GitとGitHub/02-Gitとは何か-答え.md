# Gitとは何か：答え合わせ

> 先に教材本文の4択チェックに答えてから見てください。  
> 全部正解できなくても大丈夫です。ここで確認できればOKです。

## 答え

### 1問目

- 正解：**B**
- 理由：Gitはファイルの変更履歴をコミットとして残すバージョン管理システムです。
- 間違えやすいポイント：Gitは編集ソフトやウイルス対策ではありません。

### 2問目

- 正解：**B**
- 理由：リモートはネット上の保管場所（GitHubなど）を指します。ローカルは自分のMacの中です。
- 間違えやすいポイント：「リモート＝遠い」ではなく「ネット上のコピー」と覚えるとよいです。

### 3問目

- 正解：**B**
- 理由：第10章では変更履歴をGitHubに保管し、バックアップと共有の土台を作ることが目的です。
- 間違えやすいポイント：GitHubは反応を増やす場所ではなく、主にコードやファイルの履歴管理のためのサービスです。

## もう一度確認すること

- 今日覚えること：**Git＝履歴、GitHub＝ネット上の保管、ローカル＝Macの中**
